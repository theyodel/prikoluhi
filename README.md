# Моя первая программа на C с модулями (сторонние файлы)
Как поиграть с этим говном? Легко!

### Вот инструкция:
```c
1. Скачиваем исходный код
2. Заходим в /main.c
3. Компилим файл
4. Открываем /main.exe
5. Развлекаемся!
```
