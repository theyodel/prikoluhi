# Рубрика "Эээээксперименты!"
В данной репозитории храняться мои эксперементальные мини-проекты. Вы можете скачать их и сами поиграться с кодом, возможно что-то изменить и/или улучшить. Все программы локализованы под кириллицу и латиницу с помощью библиотек `locale.h` и `windows.h`

# Как поиграть со всем этим делом?
Да легко! Вот инструкция:
```c
1. Скачиваем всю репозиторию (либо отдельную папку с нужным проектом)
2. Открываем файл main.c в редакторе кода
3. Компилируем данный файл
4. После компиляции должен появиться .exe файл (скорее всего он будет называться main.exe)
5. Запускаем .exe файл
6. Наслаждаемся
```

Если что-то пойдёт не так, смело жмите сочетание `Ctrl` + `C` (это комбинация экстренного выхода из программы в терминале)
